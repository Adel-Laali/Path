Foo bar

ما خیلی وقتا که میخوایم جک بگیم، میگیم "غضنفر". چرا؟ چون جا افتاده. چون به جای x که یه متغیره، ما اسم غضنفر رو میذاریم. اصلا خود غضنفر تبدیل شده به x و خودش نمایانگر یه متغیره.

به همین ترتیب تو دنیای برنامه نویسی خیلی وقتا از کلمات foo و bar استفاده میکنن. این کلمات خودشون مثل متغیر عمل میکنن. مثلا:

public class Animal { } public class Cat : Animal { }

به جای این کد میبینیم که بنویسن:

public class Foo { } public class Bar : Foo { }

یعنی کلاس bar از کلاس foo ارث بری کرده.
